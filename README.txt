# PelisLancer
Esta es la aplicación  de la prueba técnica
para instalarlo debe tener instalado en el ordenador PYTHON, PostgreSQL, Django
para poner usar la base de datos postgres se debe instalar el controlador o driver Psycopg2 (solo si 
se instalara la aplicacón en el sistema operativo Windows.

En caso de no encontral el driver aqui le dejo un link donde puede descargarlo http://www.stickpeople.com/projects/python/win-psycopg/index.html

Se debe crear un proyecto y ejecutar el servidor.

para correr el servidor se accede a la ubicacion del alchivo manage.py
Si no a cambiado nada estara en la carpeta Reto/PelisLancer/manage.py
Una vez en la ubicacion del mismo se ejecuta el siguiente comando: manage.py runserver
y con ese corremos el servidor.

Una vez en el servidor este corriendo se abrira la  Web Application en la intefaz de INICIO. en la cual veremos varias opciones
la primera es el boton inicio, luego el boton PELICULAS, este es para ir al login de Django, enlistar peliculas, agregar y eliminarlas.
Despues esta el Boton RENTA, este es para ir a la interfas de Django y alquilar una pelicula.
Luego esta el Boton SOBRE NOSOTROS, el cual reedirije a la pagina de Encore.
Por ultimo esta el Boton Agregar Usuario, este lleva al login de Django para agregar un nuevo usuario y ver cuales hay agregados
